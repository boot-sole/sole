# Sole

A brief Boot un-framework.


## Usage

To use this in your project, add `[sole "0.1.0-SNAPSHOT"]` to your `:dependencies`
and then require the function and use it to configure your chose sole base:

``` clojure
(require '[sole.core :refer [sole!]])

(sole! ;; FIXME!!
  )
```
